# **Can scientists create a unicorn?**

## This project investigates the fringe researchers riding the flood of animal gene-editing experimentation to create animals that don’t exist
### Inspired by [this scrollytelly](https://features.csis.org/hiddenreach/china-critical-mineral-gallium/)

## DATA CLEANING
### In regards to how I cleaned and sorted the ISAAA data I scraped, I employed a scraper that also clicked through each listed experiment, logging the gene targeted, description and other important descriptive data. I compiled the various saved csvs (separate because the site logs experiments that *were not* and *were" granted regulatory approval on separate tabs) in Google Sheets. Then, I analyzed and plotted the data in R, using ggplot and tidyverse packages.

## Then, I used Adobe Illustrator and ai2html to add in responsive images. I used ProCreate to make the artwork and the GIF, then edited the former in illustrator. 

## A new skill I learned: Procreate, ai2html and generally making a complete, time-intensive project.
